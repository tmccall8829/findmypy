FindMyPy 🥧🐍🕵️

Append a PyPI package name to findmypy.io/p/ to be redirected to any pypi package's github repo 🚀

Inspired by https://gem.wtf and similar tools ❤️

How it Works

  • Call's pypi's /json route for package info.
  • If it gets a 2xx response, it parses the github homepage URL from the payload and redirects you.
  • If it gets anything else, it'll yell at you 🙃
